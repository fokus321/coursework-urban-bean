# Urban Bean — курсовой проект

Учебный многостраничный сайт городской кофейни Urban Bean по дисциплине «Основы веб-технологий».

## Состав проекта

- `index.html` — главная страница с описанием кофейни, мультимедиа и формой предзаказа.
- `menu.html` — страница меню.
- `success.html` — страница подтверждения после отправки формы.
- `style.css` — стили, адаптивная сетка и CSS-анимации.
- `assets/` — SVG-изображения.

## Реализованные требования

- Многостраничный сайт: 3 HTML-страницы.
- Чистые HTML5 и CSS3 без библиотек и JavaScript.
- Единая шапка, меню, подвал и оформление страниц.
- Форма с обязательными полями, HTML-валидацией и редиректом на отдельную страницу.
- Адаптивность для desktop, tablet и mobile через медиазапросы.
- Доступность: `alt`, `aria-label`, skip-link, корректная структура заголовков, `prefers-reduced-motion`.
- Мультимедиа: изображения SVG и встроенное видео.
- CSS-анимации: бегущая строка, hover/focus-переходы, появление hero-блока.

## Проверка

Локальная проверка выполнена командами:

```bash
npx --yes html-validate *.html
npx --yes csstree-validator style.css
```

Ошибок не обнаружено.
